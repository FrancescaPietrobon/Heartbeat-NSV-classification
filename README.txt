The folder contains:
- Resnet: implementation of the ResNet
- LSTM_CNN: implementation of the CNN with LSTM
- AE_models: implementation of AutoEncoder with all the ML models
- final_model: implementation of the best model that will be used to make the final prediction
- test_production: code that will be used to make the final prediction
- models: folder with the AutoEncoder and SVC saved models that will be used to make the final prediction
